# WorldIMaker

World Imaker is a 3D field and scene viewer editor. 
This is a project mixed with C++, OpenGL 3.3 and SDL2.


## Guide of the editor


Camera Controls :

Rotate left : L
Rotate right : J
Move left : Q
Move right : D
Move forward : Z
Move backward : S
Up : E
Down : A


Cursor Controls :

Right : Arrow keys
Left : Arrow keys
Haut : Arrow keys
Bas : Arrow keys
Move forward : N
Move backward : B
